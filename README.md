Temperature Converter
=====================

A simple Java terminal program that converts temperatures between Fahrenheit, Celsius, and Kelvin.

Features
--------

-   Convert Fahrenheit to Celsius or Kelvin

-   Convert Celsius to Fahrenheit or Kelvin

-   Convert Kelvin to Fahrenheit or Celsius

-   Input validation to ensure only numbers are accepted

-   Option to continue converting multiple temperatures until the user quits

Getting Started
---------------

### Prerequisites

-   Java JDK installed (Java 8 or later)

-   Terminal or command line access

### Running the Program

1.  Clone or download the repository.

2.  Navigate to the project folder in your terminal.

3.  Compile the program:

`javac TemperatureConverter.java`

1.  Run the program:

`java TemperatureConverter`

1.  Follow the on-screen instructions to select a conversion and input a temperature.
